# Mobile Optimization Tasks

- [ ] Adjust milestone cards: Reduce width and height on mobile (e.g., w-[240px] h-[360px] on small screens) and ensure proper spacing.
- [ ] Opinions cards: Set responsive min-width (e.g., min-w-[300px] on mobile).
- [ ] Modals: Make them full-width on mobile (w-full), adjust grid to stack (grid-cols-1 on mobile), and reduce padding for better fit.
- [ ] General: Ensure text sizes, buttons, and spacing are mobile-friendly; add any missing responsive classes.
